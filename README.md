# Open Trivia Quiz App

## Description

A responsive quiz application built with HTML, CSS, and JavaScript using the Open Trivia DB API. Users can answer multiple-choice questions fetched dynamically from the API and view their final score.

## Features

- Fetches quiz questions from the Open Trivia DB API
- Multiple-choice questions
- Score tracking
- Responsive design
- Dynamic question loading

## Technologies Used

- HTML
- CSS
- JavaScript
- Open Trivia DB API
